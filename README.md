📌 Project Description

The Banking Information System is a Core Java application developed to simulate basic banking operations. The system allows users to register accounts, manage account details, perform transactions, transfer funds, and view account statements through a simple and user-friendly interface.

🚀 Features
User Registration
Account Management
Deposit Money
Withdraw Money
Fund Transfer
Account Statement Generation
Password Protection
Transaction History
Error Handling
Temporary Data Persistence During Session

🛠 Technologies Used
Java
NetBeans IDE
Object-Oriented Programming (OOP)
ArrayList Collection Framework

▶️ How to Run
Open the project in NetBeans.
Build the project.
Run the application.
Follow the menu options to perform banking operations.
